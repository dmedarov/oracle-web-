Automated Docker Deployment: GitHub Actions, Docker Hub, SSH, Rocky Linux, Watchtower & Oracle Cloud Free Server

This tutorial covers how to automatically build, push, and deploy Docker containers using GitHub Actions, Docker Hub, and SSH on a Rocky Linux server. Additionally, we'll set up Watchtower to keep your Docker container up-to-date and explain how to get a free server from Oracle Cloud Always Free Tier.

Prerequisites

  • A Rocky Linux or any Linux server with SSH access. (You can get one from Oracle Cloud for free, more details below)
  • Docker installed on both your local machine and the Linux server.
  • A GitHub repository with your code and a Docker Hub account for storing images.
  • GitHub Secrets set up for Docker Hub credentials and SSH access to your Linux server.

Step 1: Obtain a Free Oracle Cloud Server (Always Free Tier)

Oracle Cloud provides an Always Free Tier, which includes two virtual machines (VMs) with Linux options like Rocky Linux, Ubuntu, and more, along with several other resources.

Create an Oracle Cloud Account

Follow these steps to sign up for Oracle Cloud and get your free VM instance:

  1. Go to the Oracle Cloud Free Tier page.
  2. Click on Start for Free and fill in the registration details, including your email, password, and account details.
  3. You will need to verify your account through email and provide a valid credit card. (Oracle will not charge you unless you opt into paid services).
  4. Once verified, log in to the Oracle Cloud dashboard.

Create a Virtual Machine (VM) Instance

Once you're in the Oracle Cloud dashboard, you can create your virtual machine by following these steps:

  1. Navigate to the Compute section in the dashboard.
  2. Click on Create Instance.
  3. In the configuration options, select Always Free Eligible resources and choose your preferred operating system. Rocky Linux, Oracle Linux, and Ubuntu are all good choices for Docker hosting.
  4. Set up the instance shape to a VM.Standard.E2.1.Micro, which is free.
  5. After completing the configuration, launch the instance.
  6. Once the instance is running, you can access it via SSH using the public IP address provided.

Set Up SSH Access to Your Oracle VM

After the VM instance is created, you will need to set up SSH access to manage and deploy applications on the server:

  1. Go to the VM details page in your Oracle Cloud dashboard.
  2. Under the SSH Keys section, add your public SSH key.
  3. To generate an SSH key pair, run the following command on your local machine:
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

Copy the public key to the Oracle VM by either manually pasting it into the Oracle Cloud dashboard or using the ssh-copy-id command:

ssh-copy-id opc@

Now, you can securely connect to your Oracle Cloud VM via SSH:

ssh opc@

Step 2: Install Docker on Oracle VM (or Rocky Linux Server)

With your free Oracle Cloud server ready, follow these steps to install Docker:

sudo dnf update
sudo dnf install -y dnf-plugins-core
sudo dnf config-manager --add-repo=https://download.docker.com/linux/centos/docker-ce.repo
sudo dnf install docker-ce docker-ce-cli containerd.io

Start Docker and enable it to start on boot:

sudo systemctl start docker
sudo systemctl enable docker

Verify Docker installation:

docker --version

Step 3: Docker Hub Setup

Create a repository on Docker Hub to store your Docker images:

  1. Log in to Docker Hub.
  2. Create a new repository for your Docker images.

Next Steps

You now have an Oracle Cloud Always Free server running Docker, automatically deploying your containers with GitHub Actions and Docker Hub. Consider enhancing your setup by:

  • Adding monitoring tools like Prometheus or Grafana to track the performance of your containers.
  • Setting up notifications in your GitHub Actions workflow to alert you of failed builds or deployments.
  • Exploring Docker Compose for multi-container applications or Kubernetes for larger-scale deployments.
You’ve successfully set up an automated Docker deployment pipeline using GitHub Actions, Docker Hub, SSH, Oracle Cloud’s free server, and Watchtower!
